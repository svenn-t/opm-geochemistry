# pyGeoChem

A package with useful functions and classes for geochemical calculations

Note that the package is only expected to be used "locally". 

## How to install `pyGeoChem` package

* Navigate to top level pyGeoChem folder (*this* folder, the one containing the `pyproject.toml` file).
* Activate the Python virtual environment you wish to install the package in, e.g., using [conda](https://docs.conda.io/en/latest/) or [venv](https://docs.python.org/3/library/venv.html).
* Run `python -m pip install -e .`

The `-e` flag is not mandatory, but can be used for an [editable install](https://pip.pypa.io/en/stable/topics/local-project-installs/). This is handy if you need to make changes to the source code and want to avoid having to reinstall the package each time you do that.
